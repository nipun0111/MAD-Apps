# Mobile Application Development Lab (Semester 6)

This repository contains **all 11 mobile applications** developed as part of the **Mobile Application Development Lab** during **Semester 6**.

## 📱 About the Repository

The purpose of this repository is to showcase the practical work and hands-on experience gained throughout the lab. Each app demonstrates different concepts, tools, and techniques used in mobile app development.

## 🧩 Contents

- A total of **11 mobile applications**
- Each app is organized in its own zip file
- Covers various topics such as:
  - Data storage
  - Navigation and layouts
  - And more

## 🎯 Objective

The main objective of this lab was to:
- Understand core mobile development concepts
- Build functional mobile applications
- Gain practical development experience

## 🛠️ Technologies Used

- Android Studio
- Java for logic building
- XML for UI design
- Other libraries/tools as required by each app

## 📌 Note

Each application is independent and can be run separately.
